# 6x4表情包裁剪工具 - 设计风格指南

## 设计理念

### 核心设计原则

- **功能优先**：界面设计以实用性为核心，确保用户能够快速完成图片裁剪任务
- **简洁直观**：采用简约的设计风格，减少视觉噪音，突出核心功能
- **现代美学**：运用当代设计趋势，创造专业且友好的用户体验
- **响应式设计**：确保在各种设备上都能提供优秀的使用体验

### 颜色方案

- **主色调**：深蓝灰 (#2D3748) - 专业稳重，适合工具类应用
- **辅助色**：浅蓝 (#4299E1) - 用于强调和交互元素
- **成功色**：绿色 (#48BB78) - 操作成功反馈
- **警告色**：橙色 (#ED8936) - 提示和警告信息
- **背景色**：浅灰 (#F7FAFC) - 干净的背景色
- **文本色**：深灰 (#2D3748) - 确保良好的可读性

### 排版系统

- **标题字体**：Noto Sans SC - 现代无衬线字体，清晰易读
- **正文字体**：Noto Sans SC - 保持一致性
- **字体层级**：
  - H1: 2.5rem (40px) - 主标题
  - H2: 2rem (32px) - 副标题
  - H3: 1.5rem (24px) - 章节标题
  - Body: 1rem (16px) - 正文
  - Small: 0.875rem (14px) - 辅助信息

## 视觉效果

### 使用的核心库

1. **Anime.js** - 流畅的动画效果
2. **Pixi.js** - 高性能的2D渲染
3. **Matter.js** - 物理引擎效果（拖拽动画）
4. **Splide.js** - 图片轮播展示
5. **ECharts.js** - 数据可视化（处理统计）
6. **Shader-park** - 背景着色器效果
7. **p5.js** - 创意编程效果

### 动画效果

- **页面加载**：渐入动画，营造专业感
- **拖拽上传**：物理反馈动画，增强交互体验
- **裁剪操作**：平滑的裁剪框移动和缩放
- **按钮交互**：微妙的悬停和点击效果
- **状态切换**：流畅的界面状态转换

### 背景效果

- **主页背景**：使用Shader-park创建的动态渐变背景
- **工作区域**：简洁的纯色背景，确保专注于内容
- **卡片元素**：轻微的阴影和圆角，增加层次感

### 交互效果

- **拖拽区域**：
  - 悬停时的高亮效果
  - 拖拽时的视觉反馈
  - 文件放置后的动画确认
- **裁剪界面**：
  - 裁剪框的实时预览
  - 平滑的缩放和平移动画
  - 操作手柄的可视化反馈
- **图片列表**：
  - 缩略图的加载动画
  - 选择状态的视觉反馈
  - 删除操作的确认动画

### 响应式设计

- **桌面端** (1200px+)：
  - 三栏布局（左侧面板、中央工作区、右侧面板）
  - 大尺寸预览和详细控制
- **平板端** (768px-1199px)：
  - 两栏布局（侧边栏可折叠）
  - 适中的预览尺寸
- **移动端** (< 768px)：
  - 单栏垂直布局
  - 全屏裁剪界面
  - 触摸优化的控制元素

### 图标系统

- 使用简洁的线性图标
- 保持一致的视觉风格
- 确保在不同尺寸下都清晰可辨
- 配合文字标签使用

### 状态指示

- **加载状态**：优雅的加载动画
- **处理状态**：进度条和状态提示
- **成功状态**：绿色的确认反馈
- **错误状态**：友好的错误提示和解决方案

### 微交互设计

- **按钮点击**：轻微的缩放反馈
- **拖拽操作**：实时的位置反馈
- **裁剪调整**：即时的预览更新
- **文件上传**：进度指示和完成确认

## 用户体验细节

### 视觉层次

- 通过大小、颜色、间距建立清晰的信息层级
- 重要操作使用更醒目的视觉处理
- 辅助信息保持低调但不隐藏

### 空间布局

- 合理的留白，避免界面拥挤
- 功能区域划分清晰
- 保持视觉平衡

### 一致性

- 统一的交互模式
- 一致的视觉语言
- 标准化的组件使用

这个设计风格将确保我们的6x4表情包裁剪工具既专业又易用，为用户提供优秀的图片处理体验。
